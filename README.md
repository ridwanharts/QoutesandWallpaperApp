# QuotesandWallpaperApp
Aplikasi ini memiliki server yang dibangun sendiri menggunakan php tanpa framework, data disimpan ke firebase

## Feature Project

* Login with google account
* Firebase integration
* Save favourite user (login first)
* Admob sample
* Download image into devices
* Share image accros app
